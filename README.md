# MiniTools

A collection of lightweight productivity tools.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16 + React 19
- Tailwind CSS 4
- shadcn/ui + Radix UI
- Zustand
- TypeScript 5

## Commands

```bash
pnpm dev              # Development
pnpm build           # Build
pnpm start           # Production
pnpm lint            # Lint
pnpm audit           # Security audit
```
